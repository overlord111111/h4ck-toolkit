# 🔧 h4ck-toolkit

Kit de ferramentas para pentest, OSINT e automação de segurança ofensiva.

> ⚠️ **Aviso:** Uso educacional e testes autorizados apenas.

## Módulos

- **recon/** — Coleta de informações e footprinting
- **scan/** — Scanner de portas e serviços
- **exploit/** — Prova de conceito e exploração
- **osint/** — Busca e coleta de dados públicos
- **automation/** — Scripts de automação

## Requisitos

- Python 3.8+
- Kali Linux / Parrot OS (recomendado)
- Docker (opcional)

## Instalação

```bash
git clone https://github.com/overlord111111/h4ck-toolkit.git
cd h4ck-toolkit
pip install -r requirements.txt
```
